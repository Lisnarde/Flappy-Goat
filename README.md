# Flappy Goat

Mini jeu inspiré de Flappy Bird, mais avec une chèvre dans l'espace.  
Le but est d'esquiver des obstacle et d'aller le plus loin possible  
  
Développé en Python, en utilisant le framework pyxel.  
  
## Prérequis 

- Python 3
- La bibliothèque pyxel (`pip install pyxel`)

## Utilisation

Space : faire sauter le personnage
